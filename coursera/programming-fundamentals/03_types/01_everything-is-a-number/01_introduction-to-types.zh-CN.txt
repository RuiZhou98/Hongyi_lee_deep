此时，你已经看到了很多 C 语法并学会了它的语义， 以及如何手动执行它。 到目前为止，你看到的唯一类型是 int，如 int x = 3。 你可能想知道类型以及 int 到底是什么意思，或者就此而言，类型到底是什么。 在接下来的课程中，您将学习这些问题的答案以及 有关类型的许多其他知识。 所有这些都将建立在计算机科学的一个关键原则之上， 一切都是数字。 计算机只能对数字进行操作，因此 所有数据都必须用数字表示。 类型告诉计算机如何解释和操作数字表示。 随着本课的学习， 你将学习除了 int 之外的其他内置类型。 您还将学习隐式类型转换和显式类型转换。 以及如何制作自己的类型来表示 你需要处理的任何数据
