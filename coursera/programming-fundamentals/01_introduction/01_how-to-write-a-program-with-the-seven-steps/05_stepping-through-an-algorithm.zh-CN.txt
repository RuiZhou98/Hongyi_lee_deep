让我们简要地看一下示例算法， 以及如何自己执行其步骤。 我们不需要对编程一无所知就能完成这些步骤， 我们只需要能够做一些数学运算， 并仔细跟踪我们在做什么。 这个算法并不能完成一项真正有用的任务， 但给了我们一个简单的例子来开始。 算法说：“给定一个非负整数 N”，所以它在 N 上进行参数化。我们需要一个值 N 才能真正执行这些步骤， 所以让我们选择 N 等于二。 我们还需要一个地方来写下算法产生的输出，也 就是说，我们被告知要写下的所有内容， 我们将在这个输出框中写下来。 我们需要跟踪自己的位置， 因此我们将使用这个绿色箭头来记住我们目前正在走哪一步。 我们将从头开始。 箭头后面的步骤说要创建一个名为 X 的变量， 然后将其设置为等于 N 加二。 N 是二，二加二等于四， 所以我们要注意的是，X 最初是四。 下一步要求我们从零到 北计数。我们需要为正在计数的数字起一个名字， 这样我们就可以在其他步骤中使用它。 在这里，我们决定称之为 i。 随着时间的推移，我们将重复步骤。 我们在这里使用缩进来表示 我们计算的每个数字都要重复哪些步骤 ，并在接下来的步骤中明确注明 何时完成计数。 我们从 i 为零开始， 因为那是我们说要计算的第一个数字。 我们的下一步是写下 X 倍 i。 四次零等于零， 所以我们写下零。 下一步是让我们更新 X 的值。 它的新值是什么？ 四加零乘以二等于四 ，所以，我们会将其更新为四。 已经是四了，没关系， 我们只要把它的值保持为四即可。 现在，我们已经走到了我们所说的按我们计算的步骤的尽头。 我们需要回去数下一个数字， 所以我们将箭头移回为每个数字所做的步骤的顶部， 然后将 i 设为下一个我们要计算的数字， 在本例中为一个。 现在，我们又开始执行这些步骤了，我的价值是 1。 我现在是 X 次是多少？ 四乘一等于四， 所以我们在输出中写下数字四。 接下来，我们将再次更新 X，这一 次，X 加 i 乘以 N 的计算结果为 6， 因此我们要将 X 的值从 4 更改为 6。 我们又到了重复步骤的尽头， 所以我们需要计算下一个数字。 我们会回到顶部， 然后计算下一个值，也就是二， 然后把它当作 i 的值。 我们说从零计到 N， N 是二，那么我们算完了吗？ 在这些步骤中，我们说要包括两端， 所以当我的值为二时，我们将执行这些步骤。 所以，我们进去再做一次重复的步骤。 现在，X 次 i 的值为 12， 所以我们在输出中写下 12 然后进入下一步。 我们的下一步是再次更新 X 的值，计算 X 加 i 乘以 N， 这次我们得到 10， 所以我们将 X 的值更新为 10， 然后越过该步骤。 再一次，我们已经到了要重复的步骤的尽头， 所以我们又回到了我们说要计数的步骤。 当我们回到这里时， 我们看看我们的步骤，意识到我们已经 计算了所有应该计算的数字， 零、一和二。 所以，我们已经算完了。 现在，我们要在完成计数之后进入步骤并执行这些步骤。 这个说要写下 X 的值。X 当前的值为 10， 所以我们在输出中写下来。 现在，我们的箭在脚步的尽头。 没有什么可做的，所以我们已经完成了。 零、四、12、10 是这个算法想要我们生成的数字序列， 因为 N 等于二，我们在输出框中看到。
